Jquery Toast Plugin

Hey fellas, jquery toast plugin, as the name specifies, is the jquery plugin that lets you show beautiful toast messages to the user with haste.

It provides the user with a ton (please don't go and start counting them) of options. Please checkout the demo first and then we'll move onto explaining the available options and how to hack the plugin.

  • Show the toast at bottom left corner Fade | Sticky
  • Show the toast at bottom right corner Fade | Sticky
  • Show the toast at top left corner Fade | Sticky
  • Show the toast at top right corner Fade | Sticky

Note: Changing the position will reset the existing toast messages.
